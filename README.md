# nestjs-test

js버전으로 테스트합니다. (조금 해봣는데 너무 불편하네 ;;)

express가 아닌 fastify로 어뎁트 해서 사용합니다.

DB는 mongodb + mongoose를 사용합니다.

## Description

nest.js 의 리서치 및 learning

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

TDD는 추후.. 차근차근 ㅠㅠ

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
